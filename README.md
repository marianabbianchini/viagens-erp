# ERP de Viagens\n\nPainel para centralizar reservas aéreas, passageiros e bilhetes em um layout único.\n\n## Primeira versão\n\n- Dashboard responsivo\n- Busca por passageiro, destino, companhia ou localizador\n- Cadastro manual e envio de e-ticket\n- Visualização padronizada do bilhete\n- Impressão e geração de PDF pelo navegador\n- Atalhos para LATAM, GOL, Azul, TAP, Iberia, American Airlines e Copa Airlines\n\n## Executar localmente\n\n```bash\nnpm install\nnpm run dev\n```\n\nAcesse `http://localhost:3000`.\n
